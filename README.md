# Ruslan Huseynov – Executive Chef Portfolio

Personal portfolio website for Ruslan Huseynov, Executive Chef with 20+ years of international culinary experience.

## Features
- Single-page with inner slide-in sections (no page reloads)
- Sections: Work Experience, Skills, Achievements, Education & Certificates, Languages & Programs, Contact
- Mobile-responsive design
- Dark gold professional theme
- Lightbox for certificate/diploma images
- Pure HTML + CSS + JavaScript (no backend, no frameworks)

## Sections
- **İş Təcrübəsi** – Timeline of 20 years of career
- **Bacarıqlar** – Professional kitchen skills with animated progress bars
- **Nailiyyətlər** – International medals and culinary association memberships
- **Təhsil & Sertifikatlar** – Diplomas and certificates with photo gallery
- **Dillər & Proqramlar** – Language proficiency and software skills
- **Əlaqə** – Full contact information

## Deployment
- Static site – deploy on Render.com, GitHub Pages, Netlify, or any static host
- No build step required – just serve the files as-is

## Stack
- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Inter)
- Font Awesome 6 icons
